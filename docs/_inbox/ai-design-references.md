# AI 디자인 레퍼런스 모음

AI 시대에 디자이너와 프론트엔드의 역할이 빠르게 바뀌고 있습니다.  
단순 제작 업무는 자동화될 수 있지만, 좋은 취향과 판단력을 가진 사람은 오히려 더 큰 잠재력을 발휘할 수 있습니다.

이 문서는 AI 기반 디자인, 프론트엔드, UI 제작, 디자인 시스템, 레퍼런스 탐색에 도움이 되는 도구들을 정리한 자료입니다.

## 주요 레퍼런스

| 이름 | 설명 | 링크 |
|---|---|---|
| Pencil | IDE를 벗어나지 않고 캔버스에서 아이디어를 그리면 코드로 전환할 수 있는 도구 | https://pencil.dev/ |
| 21st.dev | AI 에이전트 레지스트리와 UI 컴포넌트 마켓플레이스. Claude Code 작업에 활용 가능 | https://21st.dev/ |
| transitions.dev | 웹 앱에 바로 적용할 수 있는 CSS 및 React 트랜지션 컬렉션 | https://transitions.dev/ |
| Refero Styles | AI 에이전트용 디자인 시스템 라이브러리. 브랜드나 URL 기반으로 `DESIGN.md` 생성 가능 | https://lnkd.in/gPUfBdnw |
| open-design | Claude Design의 오픈소스 대안. 로컬 실행 및 CLI 에이전트로 이미지와 슬라이드 생성 가능 | https://lnkd.in/gPpJq99W |
| Montage Design System | 원티드랩의 제품 디자인 시스템. 브랜드, 타이포그래피, 컴포넌트, UI 패턴 공개 | https://lnkd.in/gmeam5u9 |
| diagram-design | Claude Code용 에디토리얼 다이어그램 생성 도구. Mermaid 없이 HTML + SVG 기반으로 다이어그램 제작 | https://lnkd.in/gm6HhARd |
| Font of Web | 실제 웹사이트에서 폰트와 디자인 레퍼런스를 찾아주는 AI 기반 영감 피드 | https://fontofweb.com/ |
| Logo System | 1,000개 이상의 로고 디자인 라이브러리. Wordmark, Symbol, Animated 카테고리 제공 | https://logosystem.co/ |
| UXSnaps | 실제 앱의 UI/UX 사례와 분석을 제공하는 레퍼런스 서비스 | https://www.uxsnaps.com/ |

## 활용 관점

### 1. 디자인 아이디어를 코드로 연결

- Pencil
- open-design
- diagram-design

아이디어 스케치, 이미지, 슬라이드, 다이어그램을 빠르게 코드나 시각 자료로 전환할 때 유용합니다.

### 2. UI 컴포넌트와 디자인 시스템 참고

- 21st.dev
- Refero Styles
- Montage Design System

AI 코딩 에이전트와 함께 작업할 때 디자인 기준, 컴포넌트 구조, 브랜드 스타일을 정리하는 데 도움이 됩니다.

### 3. 인터랙션과 모션 개선

- transitions.dev

React나 웹 앱에서 자연스러운 전환 효과를 빠르게 적용할 때 활용할 수 있습니다.

### 4. 레퍼런스 탐색

- Font of Web
- Logo System
- UXSnaps

폰트, 로고, 실제 서비스 UI/UX 사례를 참고해 디자인 감각을 키우는 데 적합합니다.

## 정리

AI 디자인 도구는 단순히 화면을 자동으로 만들어주는 도구가 아니라, 좋은 취향과 판단을 더 빠르게 실험하게 해주는 도구에 가깝습니다.

디자이너와 프론트엔드 개발자의 역할은 사라지기보다, 문제 정의, 취향, 구조화, 검수, 디테일 조정 능력을 중심으로 재편될 가능성이 큽니다.
