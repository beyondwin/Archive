# Docs Inbox

Drop unclassified documents here.

Use `$archive-docs-organizer` to sort these documents into stable topic folders
under `docs/notes/`.

When organizing, move each document into an appropriate topic folder under
`docs/notes/`, and rename files only when the new name makes the content easier
to scan. Keep unedited source captures in `docs/raw/` when the source itself
needs to be preserved separately from a polished note.

The organizer should also update:

- `docs/_index/catalog.yml`
- `docs/_index/topics.yml` when a new stable topic is created
- `docs/INDEX.md`

Suggested classification order:

1. Identify the document's primary topic.
2. Reuse an existing `docs/notes/` folder if it fits.
3. Create a new topic folder only when no existing folder is appropriate.
4. Leave source-heavy or mixed-topic documents here until they can be split or
   clearly classified.
