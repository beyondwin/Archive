## Lifecycle (Phase 0 → 1 → 2)

Unchanged shape from v2.10.0 — the additions above are layered onto the same
three-phase lifecycle. The single substantive change to the loop itself is
v2.13's "Phase -1.0 three-pass argument parser" gating self-spawn.

## What ships and what does not

- **Ships**: 16 deterministic preflight checks, parallel sub-flow (P.1–P.4),
  cost accumulation per dispatch, plan-chain advance at Phase 2 Step -1,
  spec-manifest per task, decisions register accumulation, token-budget
  Resume Chain trigger.
- **Does not ship**: HTML report renderer, archive_run.sh — these were
  introduced in v2.14 but the v2.17 AgentLens cutover (post-v2.15) removed
  them in favor of AgentLens events.jsonl as the sole forensics trail.
